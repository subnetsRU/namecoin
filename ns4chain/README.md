DNS server on NodeJS for .bit domains
=================

Installation
------------

```sh
# npm install native-dns
# npm install json-rpc2
# npm install fs
# npm install util
# npm install sprintf-js
# npm install insubnet
```

Usage: node dns_serv.js [options]

Options:
* -h, --help
* -d, --debug <none|log|cli|full>
* -l, --listen <IP>
* -p, --port <PORT>
* -t, --ttl <NUMBER>
* -r, --recursion
